

A simple **Java Swing GUI application** that converts different units such as length, weight, and temperature.
Users can enter a value, select the conversion type, and instantly see the result.

---

## 📌 Features,

* Convert **Kilometer to Meter
* Convert **Meter to Centimeter**
* Convert **Kilogram to Gram**
* Convert **Gram to Kilogram**
* Convert **Celsius to Fahrenheit**
* Convert **Fahrenheit to Celsius**
* Simple and clean **Java Swing interface**

---

## 🛠️ Technologies Used

* Java
* Java Swing (GUI)

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/jitendrakumar006/java-unit-converter.git
```

2. Compile the program

```
javac UnitConverter.java
```

3. Run the program

```
java UnitConverter
```

## 📷 Application Overview

The application contains:

* Input field to enter the value

* Convert button
